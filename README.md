# 🎬 Anime Social Media Platform

A full-stack anime-based social platform where users can share posts, react, and avoid spoilers intelligently based on their watchlist.

---

## 🚀 Features

- 🧠 **Smart Spoiler Protection**
  - Automatically blurs spoiler content
  - Uses watchlist + completion status for personalized filtering

- 📺 **Anime Watchlist System**
  - Track anime as:
    - Watching
    - Completed
    - Plan to Watch

- 💬 **Discussion Threads**
  - Engage in conversations on posts

- ❤️ **Reaction System**
  - React with:
    - Peak
    - Sad
    - Shock
    - Mindblown
    - Love

- 🔐 **Authentication**
  - Secure JWT-based login/signup

- 🖼 **Image Upload**
  - Upload images using Cloudinary

---

## 🧠 Unique Idea

Unlike traditional social platforms, this app prevents spoilers dynamically.

👉 Users will NOT see spoiler content unless:
- They have completed the anime  
- OR they manually reveal it  

This creates a **personalized spoiler-safe experience**.
